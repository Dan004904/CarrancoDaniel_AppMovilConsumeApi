# Aplicación Móvil CRUD (Maestro-Detalle)

## 🚨 Aviso Importante sobre el Rendimiento Inicial (API)

El backend de esta aplicación se encuentra alojado utilizando el **plan gratuito de [Render](https://render.com/)**. 

Por las políticas de esta plataforma, cuando el servicio web no recibe tráfico durante un periodo de tiempo, entra en un estado de **hibernación** (spin-down) para ahorrar recursos. 

**¿Cómo afecta esto a la aplicación?**
* Al abrir la aplicación por primera vez después de un periodo de inactividad, la primera petición para obtener la lista de usuarios puede tardar entre **30 segundos y 1 minuto** en completarse mientras el servidor se "despierta".
* Si al iniciar la app no ves a los usuarios inmediatamente, **no es un error de la aplicación**. Por favor, espera unos segundos.
* Una vez que el servidor se ha reactivado, todas las operaciones posteriores (Leer, Crear, Actualizar y Eliminar) funcionarán con total normalidad y fluidez.
